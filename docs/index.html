API Docs

Projects

  • List projects

    Get a list of all projects the authenticated user is allowed to access

  • Single project

    Get detailed information about a single project

Data groupings

  • Single data grouping

    Get a single data grouping and its contributions.

  • My contributions

    Get all contributions of the authenticated user.

  • All contributions in a project

    Get all contributions the authenticated user is allowed to access in the project.

Contributions

  • Single contribution

    Get a single contribution from a data grouping.

  • Add contribution

    Add a new contribution to a project.

  • Update contribution

    Update an existing contribution.

  • Delete a contribution

    Delete an existing contribution.

Comments

  • List comments

    Get a list of all comments to a contribution.

  • Add comment

    Add a comment to a contribution.

  • Delete comment

    Delete a comment from a contribution.
