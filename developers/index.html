Developers

GeoKey is 100% open source. You can deploy it on your own servers, extend it or build an app on top of the API.

Develop an application on top of the API

Whether you want to build a smart phone app to collect data, visualise your data on a web map or want to build a bridge to your GIS — the REST API provides access to all information you need.
